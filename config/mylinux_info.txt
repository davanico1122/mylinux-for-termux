═══════════════════════════════════════════════════════════════════════════════
                         QUICK START GUIDE
═══════════════════════════════════════════════════════════════════════════════

[STARTING MYLINUX]
  • From Termux: Type 'mylinux' to enter the environment
  • Auto-start: The interactive menu will launch automatically

[BASIC COMMANDS]
  • menu      : Show interactive menu system
  • update    : Update system packages and tools
  • scan      : Quick network scan (nmap -sn)
  • speedtest : Run internet speed test
  • fontlist  : Show available figlet fonts
  • exit      : Exit MyLinux environment

[FILE MANAGEMENT]
  • ll    : List files in detail (ls -alF)
  • la    : List all files including hidden (ls -A)
  • tree  : Show directory structure (tree -L 3)
  • extract: Extract compressed files (supports zip, tar, rar, etc)

[PACKAGE MANAGEMENT]
  • install <package> : Install new package (sudo pacman -S)
  • search <keyword>  : Search for packages (pacman -Ss)
  • remove <package>  : Remove package (sudo pacman -R)

═══════════════════════════════════════════════════════════════════════════════
                         CUSTOMIZATION GUIDE
═══════════════════════════════════════════════════════════════════════════════

[CHANGING BANNER]
  1. Type 'menu' to open interactive menu
  2. Select option [8] 🎨 Customize Banner
  3. Choose from available fonts or use default

[ADDING ALIASES]
  Edit ~/.bashrc and add new aliases:
    alias short='long command'
  
  Example:
    alias ports='netstat -tuln'

[CHANGING THEME]
  1. Install preferred theme engine (e.g., oh-my-zsh)
  2. Configure in ~/.bashrc or ~/.zshrc
  3. Recommended themes for mobile:
     • termux-style
     • powerlevel10k

═══════════════════════════════════════════════════════════════════════════════
                         TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

[COMMON ISSUES]
  • Internet connection not working:
    - Run: pacman -Syu --noconfirm
    - Run: pacman -S networkmanager

  • Storage full:
    - Clean package cache: sudo pacman -Sc
    - Remove unused packages: sudo pacman -Rns $(pacman -Qtdq)

  • Menu not starting automatically:
    - Ensure ~/.bashrc contains: 'mylinux-menu' at the end

[MAINTENANCE COMMANDS]
  • Update all: sudo pacman -Syu && pip install -U lolcat speedtest-cli
  • Clean cache: sudo pacman -Sc
  • Fix broken: sudo pacman -Syu --overwrite '*'

                         UNINSTALLATION
═══════════════════════════════════════════════════════════════════════════════

To completely remove MyLinux:
  1. Exit MyLinux environment
  2. In Termux, type: uninstall-mylinux
  3. Confirm uninstallation

Note: This will remove all MyLinux files and configuration.
